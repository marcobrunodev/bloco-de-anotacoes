# Dicas do Terminal

Vou listar alguns que comando que estamos aprendendo:

- `cd` (navega entre pastas)
  Exemplo para entrar dentro de uma pasta:

```
cd nomeDaPasta
```

Exemplo para sair de uma pasta:

```
cd  ..
```

- `mkdir` (cria pastas)
  Exemplo de como criar uma pasta

```
mkdir nomeDaPasta
```
